# Як задеплоїти грамматичну перевірку (Cloudflare Worker)

Це безкоштовно (щедрий free tier), без картки для реєстрації, займе ~10 хвилин.

## 1. Отримай API-ключ OpenAI
1. Зайди на https://platform.openai.com/api-keys
2. Створи акаунт (якщо ще немає) і поповни $5 — цього вистачить на дуже довго: `gpt-4o-mini` коштує копійки (~$0.15 за мільйон токенів, одна перевірка речення — це десь 0.01–0.05 центу).
3. Створи новий Secret Key, скопіюй його (він показується один раз).

## 2. Створи Cloudflare Worker
1. Зайди на https://dash.cloudflare.com → зареєструйся (безкоштовно).
2. У меню зліва: **Workers & Pages** → **Create** → **Create Worker**.
3. Дай йому назву, наприклад `english-grammar-checker`.
4. Натисни **Deploy** (спочатку задеплоїться дефолтний "Hello World" код — це нормально).
5. Натисни **Edit code**.
6. Видали весь код, що там є, і встав вміст файлу `worker.js`.
7. Натисни **Deploy** (або **Save and Deploy**).

## 3. Додай секретний ключ
1. У воркері перейди на вкладку **Settings** → **Variables and Secrets**.
2. Додай змінну:
   - Name: `OPENAI_API_KEY`
   - Value: (встав ключ, який скопіювала в кроці 1)
   - Тип: **Secret** (не Plaintext!)
3. Збережи. Воркер сам передеплоїться з новим секретом.

## 4. Скопіюй URL воркера
Він виглядає так:
```
https://english-grammar-checker.ТВІЙ-SUBDOMAIN.workers.dev
```
Знайдеш його на сторінці воркера, зверху.

## 5. Встав URL в index.html
Відкрий `index.html`, знайди рядок:
```js
const GRAMMAR_WORKER_URL = 'https://YOUR-WORKER-NAME.YOUR-SUBDOMAIN.workers.dev';
```
Заміни на свій реальний URL з кроку 4.

## 6. (Рекомендовано) Обмеж CORS
У `worker.js` зараз стоїть:
```js
'Access-Control-Allow-Origin': '*',
```
Це дозволяє будь-якому сайту користуватись твоїм воркером (і твоїм OpenAI-бюджетом). Краще заміни `*` на свій точний GitHub Pages домен, наприклад:
```js
'Access-Control-Allow-Origin': 'https://твій-юзернейм.github.io',
```
і передеплой воркер (Edit code → встав → Deploy).

## 7. Заливай на GitHub Pages як завжди
`index.html` більше нічого не запитує в LanguageTool — тепер він ходить у твій воркер, який реально розуміє контекст, часи, прийменники і навіть повністю безглузді речення.

## Перевірка
Відкрий сайт, скажи навмисно щось із помилками (як у твоєму скріншоті) — тепер має підсвітити прийменники, часи і структуру речення, а не писати "Жодної помилки — чудово!" на суцільну плутанину.
