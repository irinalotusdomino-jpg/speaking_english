# Як задеплоїти грамматичну перевірку (Cloudflare Worker + Workers AI)

Повністю безкоштовно, картка не потрібна взагалі. Займе ~10 хвилин.

Це працює через **Cloudflare Workers AI** — вбудований у Cloudflare хостинг моделей (10 000 "нейронів" на день безкоштовно, цього вистачить на десятки-сотні перевірок щодня). Ніякого окремого API-ключа заводити не треба.

## 1. Створи Cloudflare Worker
1. Зайди на https://dash.cloudflare.com → зареєструйся, якщо ще немає акаунту (без картки).
2. Зліва: **Workers & Pages** → **Create** → **Create Worker**.
3. Дай назву, наприклад `english-grammar-checker`.
4. Натисни **Deploy** (спочатку задеплоїться дефолтний "Hello World" — це нормально).
5. Натисни **Edit code**.
6. Видали весь код і встав вміст файлу `worker.js`.
7. Натисни **Deploy** (або **Save and Deploy**).

## 2. Увімкни Workers AI для цього воркера
1. На сторінці воркера перейди у вкладку **Settings** → **Bindings**.
2. Натисни **Add binding** → обери **Workers AI**.
3. Variable name: обов'язково `AI` (саме так це названо в коді — `env.AI`).
4. Збережи. Воркер передеплоїться автоматично.

Це все — жодних ключів, жодних оплат.

## 3. Скопіюй URL воркера
Виглядає так:
```
https://english-grammar-checker.ТВІЙ-SUBDOMAIN.workers.dev
```
Знайдеш зверху на сторінці воркера.

## 4. Встав URL в index.html
Відкрий `index.html`, знайди рядок:
```js
const GRAMMAR_WORKER_URL = 'https://YOUR-WORKER-NAME.YOUR-SUBDOMAIN.workers.dev';
```
Заміни на свій реальний URL з кроку 3.

## 5. (Рекомендовано) Обмеж CORS
У `worker.js` зараз стоїть:
```js
'Access-Control-Allow-Origin': '*',
```
Це дозволяє будь-якому сайту юзати твій воркер (і твій денний ліміт нейронів). Краще заміни `*` на свій точний GitHub Pages домен:
```js
'Access-Control-Allow-Origin': 'https://твій-юзернейм.github.io',
```
і передеплой (Edit code → встав → Deploy).

## 6. Заливай на GitHub Pages як завжди
`index.html` більше нічого не питає в LanguageTool — тепер ходить у твій воркер, який реально розуміє контекст, часи, прийменники і навіть повністю безглузді речення.

## Якщо забракне безкоштовного ліміту
10 000 нейронів/день — це орієнтовно кілька сотень перевірок речень на день з моделлю в коді (`@cf/meta/llama-3.3-70b-instruct-fp8-fast`). Якщо колись впреться в ліміт:
- або зменш модель на легшу — в `worker.js` заміни `MODEL` на `'@cf/meta/llama-3.1-8b-instruct'` (менш точна, але швидша й "дешевша" по нейронах),
- або просто почекай до наступного дня (ліміт скидається щодня о 00:00 UTC).

## Нова функція: "Слухати і перекладати"
Внизу сторінки з'явився блок, куди можна вставити текст або посилання (URL) — англійською чи українською:
- **🔊 Прослухати як є** — просто озвучує вставлений текст (визначає мову автоматично).
- **🇺🇦→🇬🇧** — перекладає українською→англійською і одразу озвучує переклад.
- **🇬🇧→🇺🇦** — перекладає англійською→українською і озвучує.

Якщо вставити посилання — воркер сам завантажить сторінку і витягне текст з неї.

Це працює через той самий воркер, тому окремого деплою не треба — просто онови код у `worker.js` (кроки 1 і 6 нижче лишаються ті самі).

⚠️ Озвучення української залежить від голосів, які є в браузері/ОС користувача. У Chrome на Windows/Android зазвичай є `uk-UA` голос, на iOS Safari — не завжди. Якщо українського голосу не знайдено, браузер прочитає текст голосом за замовчуванням (може звучати з акцентом).

## Перевірка
Відкрий сайт, скажи навмисно щось із помилками (як у твоєму скріншоті) — тепер має підсвітити прийменники, часи і структуру речення, а не писати "Жодної помилки — чудово!" на суцільну плутанину.
